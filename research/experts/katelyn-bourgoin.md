## Why chosen

Translates peer-reviewed buyer psychology into B2B content frameworks. "Why We Buy" newsletter: 63K subscribers, $1M revenue in 2024. 4x founder. Direct mapping to researcher's psychology background.

## Key frameworks

| Framework | Summary |
|---|---|
| JTBD | People hire products to make progress — describe the job, not the feature |
| Trigger events | Content around moments that start the buying journey |
| Rumpelstiltskin Effect | Naming a problem gives the reader power over it |
| Ownable idea | 5-question framework: aspiration → problem → lens → revelation → directive |
| Clarity calls | 1 interview > 1,000 surveys for content language |
| Mere exposure | Consistency beats virality |

## Psychological triggers

Identity threat (Joshua Bell), curiosity gap, loss aversion, narrative transportation, Rumpelstiltskin Effect, Expert Paradox

## B2B SaaS application

- Write trigger event posts for passive lookers (stages 1–2 of buying timeline)
- Coin a named problem; use in every awareness post
- Run 5 clarity calls → extract exact buyer language for hooks
